Regular Expression

URL Test

Pengisian URL

Silakan masukkan URL lengkap dengan protokol, seperti 'https://www.contoh.com'. Pastikan Anda menyertakan '.co' atau '.com' di akhir URL agar input dianggap valid.

Minimal 10 kata

Pengisian Min 10 kata

Harap masukkan setidaknya sepuluh kata dalam input ini. Mohon pastikan bahwa input Anda memiliki jumlah kata yang memadai.

Email Test

Pengisian Email

Silakan masukkan alamat email Anda. Pastikan formatnya benar, seperti 'nama@example.com'.

Password Test

Pengisian Password

Mohon masukkan kata sandi Anda. Kata sandi harus terdiri dari minimal 8 karakter, setidaknya satu huruf besar, satu huruf kecil, satu angka, dan satu karakter istimewa.

Nama Ekstensi File Test

Nama Ekstensi File Test

Silakan masukkan nama ekstensi file. Pastikan nama ekstensi file yang Anda masukkan sesuai dengan format yang diharapkan, seperti '.jpg', '.pdf', dll.
